### 系统特性

#### 多用户支持
- 每个用户独立账户（初始资金 ¥10,000）
- 独立持仓和交易记录
- Token 认证，数据隔离

#### 股票数据
- **优先**: Tushare Pro 实时行情（需要配置 Token）
- **备用**: 模拟数据（无 Token 时自动切换）
- **全市场扫描**: 沪深两市约5000只股票筛选

#### 技术策略
- **均线系统**: MA5, MA10, MA20, MA30
- **MACD**: 金叉/死叉信号
- **RSI**: 超买/超卖判断
- **量比**: 成交量放大检测
- **综合评分**: 多因子加权评分（0-100）

#### 交易模拟
- **买入**: 支持加仓（已有持仓）
- **卖出**: 部分或全部卖出
- **费用**: 佣金（0.03%，最低5元）+ 印花税（卖出0.1%）
- **盈亏**: 实时计算，百分比显示

#### 定时任务
- **交易日**: 周一至周五
- **交易时间**: 9:00-15:00
- **扫描间隔**: 每30分钟（0分和30分）
- **手动触发**: 支持立即扫描

# 2026-03-26 补丁日志

## Bug修复1: Railway部署失败 - pandas版本不兼容
- **问题**: pandas==2.0.3 在 Python 3.12 下编译失败（pkg_resources 找不到）
- **修复**: pandas 升级到 2.2.2，numpy 升级到 1.26.4，添加 setuptools>=65.0.0

## Bug修复2: 登录后无限刷新死循环
- **问题**: 后端 `/` 路由尝试从 Authorization 请求头读 Token，但浏览器跳转不携带该头，导致始终重定向回登录页，login.html 检测到 localStorage 有 Token 再跳主页，死循环
- **修复**:
  - 后端 `/` 路由直接返回 `index_auth.html`，不再判断 Token
  - `index_auth.html` 前端 JS 检测 localStorage Token，无 Token 直接跳到 `/login`
  - `login.html` 检测到 Token 直接跳主页，不再重复调用 API 验证
- **文件**: `app_auth.py`, `index_auth.html`, `login.html`

## Bug修复3: 推荐股票显示undefined，价格不显示，详情功能缺失
- **问题**: 数据库字段名 stock_code/stock_name，前端访问 item.code/item.name，不匹配；showStockDetail 是空壳函数
- **修复**:
  - `database.py` load_suggestions 函数增加字段映射 stock_code→code, stock_name→name
  - 前端股票卡片加空值保护，显示买入价/止损价/目标价
  - showStockDetail 改写为完整弹窗，展示全部价格分析
- **文件**: `database.py`, `index_auth.html`
