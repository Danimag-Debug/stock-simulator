# 2026-04-01 开发日志

## 持仓卖出建议功能
1. 后端 `engine_db.py` 新增 `_calc_sell_advice()` 函数，为每只持仓股票实时计算卖出建议
   - 优先使用 ATR 动态止损法（technical_analyzer 模块）
   - 降级使用阶梯止损法：按持仓盈亏分5档（大赚/中赚/小赚/小亏/大亏）
   - 返回：action、action_color、reason、stop_loss、target_price、upside_pct、downside_pct、risk_reward
2. `get_portfolio_snapshot()` 调用 `_calc_sell_advice()`，在每个持仓记录中附加 `sell_advice` 字段
3. 前端持仓表格新增"卖出建议"列，显示操作建议、止损价、目标价

## 股票查询功能（v2 - 全市场覆盖）
1. 新增 `stock_name_list.py` 静态股票列表文件（5417只沪深股票代码-名称映射）
2. `_load_stock_name_cache()` 改为三层层级加载：静态文件 → 东方财富API → 模拟池
3. `search_stock_by_name()` 在全量缓存中搜索，支持精确匹配和模糊匹配
4. `query_stock_score()` 使用缓存做名称搜索，不再依赖 Tushare pro.stock_basic
5. 新增 `GET /api/stock/search?keyword=xxx` 模糊搜索 API（返回多个匹配，最多50个）
6. 前端查询框增加自动补全下拉列表（输入即搜索、键盘上下选择、Enter确认）
7. 停牌/退市股票返回友好提示而非空结果
