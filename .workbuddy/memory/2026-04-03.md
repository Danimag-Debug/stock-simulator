# 2026-04-03 工作日志

## 修复推荐股票多样性问题（commit: cd987e6）

**问题**：每次扫描推荐的总是相同几只股票（茅台、宁德时代等大盘股）

**根因分析**：
1. 东方财富候选池只取4页（约2000只），均为成交额最高的大盘股
2. 行业热度评分是静态确定性映射，导致同一股票每次得分几乎一样
3. 最终排序随机扰动太小（0~1.5分），高分股票固定排最前

**修复内容**：
- 东方财富数据：4页→10页（2000→约5000只），覆盖更多中小盘
- 随机扰动：0~1.5分 → 0~5.0分，评分相近的股票都有机会出现
- 推荐轮换机制：上次已推荐的股票降权8分，强制引入新面孔
- 日志增加新面孔/重复只数统计
